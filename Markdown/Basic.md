## bold

to **bold** text surround it with two asterisk ( \* ) or double underscore ( \_ \_ )

```markdown
**text**
**text**
```

<hr>
## italics

to _italics_ text surround it with one asterisk ( \* ) or one underscore ( \_ )

```markdown
_text_
_text_
```
